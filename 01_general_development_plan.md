# План разработки лендинга для парсинга цен интернет-магазинов

## 📋 Общая архитектура проекта

### Технологический стек
- **Frontend**: React 18+, React Router, Axios, Recharts/Chart.js
- **Backend**: Node.js, Express.js, MongoDB
- **Парсинг**: Puppeteer Cheerio

---

## 🎯 Этапы разработки

### **Этап 1: Проектирование и подготовка (2-3 дня)**
- Создание wireframes и mockups
- Определение структуры базы данных
- Выбор целевых интернет-магазинов для парсинга
- Настройка Git репозитория

### **Этап 2: Backend разработка (5-7 дней)**
- Настройка Node.js/Express сервера
- Реализация REST API endpoints
- Интеграция базы данных (модели, схемы)
- Разработка модуля парсинга с rate limiting
- Реализация аналитического движка для обработки данных (SQL запрос)
- Настройка CRON задач для периодического парсинга

### **Этап 3: Frontend разработка (5-7 дней)**
- Настройка React проекта с маршрутизацией
- Блок 1: Строка поиска (Google-style)
- Блок 2: Зона графиков и аналитики
- Блок 3: Популярные товары и площадки
- Блок 4: Футер с контактами
- Реализация адаптивного дизайна
- Интеграция с Backend API

### **Этап 4: Интеграция и стилизация (3-4 дня)**
- Применение цветовой схемы (зеленый акцент, белый/серый фон)
- Анимации и микроинтерактивность
- Оптимизация производительности

### **Этап 5: Тестирование и отладка (3-4 дня)**
- Integration тесты
- Исправление багов

## 📊 Ключевые функции

### 1. **Поисковая система**
- Автодополнение запросов
- Поиск по категориям и ключевым словам
- История поисковых запросов

### 2. **Аналитика и графики**
- Динамика цен за период (линейный график)
- Сравнение цен между магазинами (столбчатая диаграмма)
- Распределение по категориям (круговая диаграмма)
- Рекомендованная цена на основе анализа

### 3. **Популярные товары**
- Карточки топ-10 товаров
- Индикатор изменения цены (рост/падение)
- Ссылки на магазины
- Фильтрация по категориям

### 4. **Футер**
- Социальные ссылки (GitHub, Email)
- Краткая информация о проекте
- Политика конфиденциальности
