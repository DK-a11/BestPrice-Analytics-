# PriceParser - Frontend

Веб-приложение для мониторинга цен из популярных интернет-магазинов.

## Технологии

- React 18
- Vite
- TailwindCSS
- Recharts
- Axios

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

## Сборка

```bash
npm run build
```

## Структура проекта

- `src/components/` - React компоненты
- `src/services/` - API клиенты
- `src/hooks/` - Custom React hooks
- `src/utils/` - Утилиты
- `src/styles/` - Глобальные стили
