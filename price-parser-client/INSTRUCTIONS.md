## 🎨 Что вы увидите

### Блок 1: Поиск (Google-style)
- Центрированная строка поиска с зеленым акцентом
- Автодополнение при вводе (работает с задержкой 300ms)
- История последних 10 поисков (хранится в localStorage)
- Dropdown с подсказками и категориями
- Клавиатурная навигация (стрелки вверх/вниз)

### Блок 2: Аналитика
- **График динамики цен** - линейный график изменения цен во времени
- **Сравнение магазинов** - столбчатая диаграмма текущих цен
- **Инсайты** - средняя цена, лучшее предложение, изменение за неделю
- Переключатель периодов (7, 30, 90 дней)

### Блок 3: Популярные товары
- Сетка карточек товаров (6 товаров)
- Фильтры по категориям (Все, Электроника, Одежда и т.д.)
- Индикатор изменения цены (зеленый ↓ для падения, красный ↑ для роста)
- Badge "Лучшая цена" для выгодных предложений
- Секция популярных площадок (Wildberries, Ozon, Яндекс.Маркет, DNS)

### Блок 4: Футер
- 4 колонки: О проекте, Навигация, Категории, Контакты
- Социальные иконки (GitHub, Telegram, Email)
- Копирайт и legal links
- Анимированное сердечко ❤️

Когда backend будет готов, обновите файл `.env`:


- `searchSuggestions()` - автодополнение

- `getPriceHistory()` - история цен

- `getPopularProducts()` - популярные товары

## 🐛 Troubleshooting

**Проблема:** Ошибки при установке зависимостей  
**Решение:** Попробуйте: `npm install --legacy-peer-deps`

**Проблема:** Порт 5173 занят  
**Решение:** Измените порт в `vite.config.js` или закройте процесс на порту

**Проблема:** TailwindCSS не работает  
**Решение:** Проверьте импорт в `src/index.css`: `@tailwind base;`

**Проблема:** Графики не отображаются  
**Решение:** Убедитесь что установлен `recharts`: `npm install recharts`
